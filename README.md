# Introduction

This repository contains a set of vanilla JavaScript UI components designed for modern web applications that need rich interaction without heavy frameworks.

---

## Included Components

### DataTable
An interactive data table component with:
- AJAX‑driven data loading
- Client‑side sorting and searching
- Pagination
- Single and multi‑row selection
- Keyboard navigation and focus management
- Screen‑reader friendly markup patterns

### Modals
An accessible modal dialog system featuring:
- Dynamic AJAX content loading
- Focus trapping and automatic focus restoration
- ESC‑key dismissal
- Optional draggable behavior
- Expand / compress support
- Designed for forms and dialogs

### Tabs
A keyboard‑accessible tab interface that provides:
- AJAX‑loaded tab content
- Arrow‑key navigation
- Automatic ARIA roles and attributes
- Horizontal and vertical tab layouts
- Progressive enhancement over semantic HTML

---

## Key Features

- **No external JavaScript dependencies**
- **Framework‑agnostic** (works with server‑rendered HTML)
- **Accessibility‑first** (WCAG‑aligned behavior)
- **Keyboard‑friendly** by default
- **Progressive enhancement**, not replacement

---

## Typical Use Cases

- Admin dashboards
- Internal web apps
- Legacy systems needing modern UI behavior without JavaScript frameworks

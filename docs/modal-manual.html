Modal Component — Developer Manual

The ModalExt component provides dynamic, AJAX-loaded modal windows with keyboard accessibility, optional drag support, automatic focus handling, and expand/compress capabilities. It is used throughout the UI to display forms, dialogs, and content overlays.

Introduction

The modal system enables developers to attach modal behavior to any element by giving it the modal class and providing data-url and data-action attributes. When activated, the modal fetches content from the server, inserts it into a styled overlay window, and manages user interaction.

Usage Example

<div class="modal" data-url="/ajax/editUser" data-action="edit">
  Edit User
</div>

After calling setupComponents(), clicking the element automatically loads the modal content and displays it.

Initialization

Object.assign(modalElement, ModalExt);
modalElement.init();

Features

  • Dynamic AJAX loading
  • Draggable modal windows
  • Escape key closes modal
  • Automatic focus and accessibility support
  • Expandable full-screen mode

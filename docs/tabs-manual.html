Tabs Component — Developer Manual

The TabsExt component provides keyboard-accessible, AJAX-powered tab navigation for switching between different content areas without reloading the page.

Introduction

Tabs are defined using elements with the tab class and a data-url attribute. Selecting a tab triggers an AJAX request that loads content into a paired tabContent container. Tabs support arrow-key navigation and automatic ARIA labeling.

Usage Example

<div class="tabs">
  <span class="tab" data-url="/ajax/info">Info</span>
  <span class="tab" data-url="/ajax/settings">Settings</span>
</div>
<div class="tabContent"></div>

When a tab is clicked, the target content is fetched via AJAX and rendered inside the content container.

Initialization

Object.assign(tabElement, TabsExt);
tabElement.loadTab();

Features

  • Automatic AJAX content loading
  • Keyboard arrow navigation
  • Accessible ARIA roles and focus handling
  • Vertical and horizontal tab layouts supported
